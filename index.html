SECURITY VALUATOR [Personal Project]

Languages/packages used: Python, Django, Pandas, ScraPy

GitHub: https://github.com/sebkeil/SecurityValuator

Why this project? With this project I aimed to apply my knowledge of Python to a create a tool that I could use myself when looking for stocks to invest in. The tool is based on a common financial evalution method called 'Discounted Cash Flow Method', which basically aims to find to the intrinsic value of a stock by analyzing and projecting Free Cash Flows and other relevant financial data. The website consists of several sub-tools. First, it includeswebscraper that fetches relevant data from YahooFinance (written in Scrapy) and displays it as pandas dataframes. Second, it has a web-based calculator for the Weighted Average Cost of Capital (WACC), which is a metric to the DCF methodology. Third, the DCF calculator applies a series of projections and calculations to arrive at the fair value of the given company. For those who are interested, the code roughly follows the method outlined in this video.

The images below show some impressions on what the website has to offer. Please note that it is currently not running live on a server.

What are the next steps? I personally like to use this tool to assist me in analyzing stocks. The next step is to improve the usability of this tool by integrating the four steps into one (i.e. the user only has to fill in the stock ticker and his basic assumptions and receives a full analysis). To complement the output, graphs will be added. After this is achieved, a login-system needs to be implemented so that users may collect their analyzed securities in a personalized portfolio.

ESTIMATING AIRBNB PRICES IN AMSTERDAM [University Group Project]

Languages/packages used: Python, ScikitLearn, Pandas, TensorFlow

GitHub: https://github.com/sebkeil/Group20-VU/tree/master/amsterdam-airbnb

Report: Click HERE

About this project. In this machine learning project we experimented with a variety of different ML algorithms in the attempt to accurately predict AirBnb prices in Amsterdam. We also used a variety of methods to optimize hyperparameters, namely Grid Search CV, Randomized CV and Evolutionary Search. The main result of this investigation was that ensemble methods like Gradient Boosting Regressor, Extra Trees Regressor and Random Forest Regressor outperformed other ML methods.

My role in this project. As I was eager to learn how to implement key ML methods in practice, I took the initiative in doing the programming for this project. This strongly improved my ability to do data science with Python and libraries like ScikitLearn, pandas, TensorFlow and TPOT. I also helped my collegues to do research by engaging the existing literature on regression problems.

MASTERING THE GAME OF SCHNAPSEN WITH ENSEMBLE VOTING [University Group Project]

Languages/packages used: Python, ScikitLearn, R

Report (incl. Code): Click HERE

About this project. The task was to build an intelligent system that could succeed in the game of Schnapsen against a series of benchmark agents. Our team designed a number of bots and had them compete with the benchmark models. We observed that ML-based bots seemed to perform best, with our ensemble bot consistently outperforming all the other agents.

My role in this project. I helped in the design of the bots and was in charge of conducting the statistical analysis, using the statistical package R. I also engaged with the existing academic literature on agent design. This project taught me a lot on how to approach the complex task of designing, engineering and testing intelligent agents.
