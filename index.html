SECURITY VALUATOR [Personal Project]

Languages/packages used: Python, Django, Pandas, ScraPy

GitHub: https://github.com/sebkeil/SecurityValuator

Why this project? With this project I aimed to apply my knowledge of Python to a create a tool that I could use myself when looking for stocks to invest in. The tool is based on a common financial evalution method called 'Discounted Cash Flow Method', which basically aims to find to the intrinsic value of a stock by analyzing and projecting Free Cash Flows and other relevant financial data. The website consists of several sub-tools. First, it includeswebscraper that fetches relevant data from YahooFinance (written in Scrapy) and displays it as pandas dataframes. Second, it has a web-based calculator for the Weighted Average Cost of Capital (WACC), which is a metric to the DCF methodology. Third, the DCF calculator applies a series of projections and calculations to arrive at the fair value of the given company. For those who are interested, the code roughly follows the method outlined in this video.

DEEP LEARNING PROJECTS [University Individual Projects]

Languages/packages used: Python, PyTorch, NumPy, Matplotlib

During the course 'Deep Learning' at the VU Amsterdam, we had to complete a challenging assignment each week that is worth mentioning here. The assignments, in conjunction with the lecture part of the course, gave me a deep intuition into the workings of neural networks and equipped me with the skills to apply those practice. These were some of the topics that were covered in the assignments:

Week 1: Implement two neural networks (MLPs) from scratch, one using pure python and using NumPy. We had to compute relevant derivatives by hand and the main challenge laid in applying tensor operations and backpropagation using simple looping structures. After implementation, the MLPs had to be trained and tested in a experimental context, using synthetic data as well as the MNIST dataset.

Week 2: Implement an autodiff engine, inspect an advanced autodiff engine and get familiar with PyTorch. We had to automate the process of computing derivatives and backpropagation, as well as understand the computation graph of neural networks. This provided deep insights on the underlying working of popular deep learning libraries like PyTorch.

Week 3: Implement Convolutional Neural Netoworks (CNNs) in PyTorch. We had work with a variety of CNN architectures and inspect their performance in an experimental setting, using tasks related to image classification.

Week 4: Implement Recurrent Neural Networks (RNNs) in PyTorch. We experimented with a number of different models, including Elman RNNs and LSTMs, which were applied to solve sequential tasks (e.g. sentiment analysis).

Week 5: Implement Variational Autoencoder (VAE) and Generative Adverserial Network (GAN) in PyTorch. We learned on and reconstructed the MNIST dataset, and the experimented with changing the latent space (for VAE) and employ different convolutional architectures (for GAN).

COMPUTER VISION PROJECTS [University Group Projects]

Languages/packages used: Python, Open CV, NumPy, Matplotlib

GitHub: https://github.com/sebkeil/ComputerVision-Group

As I took the course 'Computer Vision 1' at the University of Amsterdam (UvA), we had to complete a complex programming assignment each week, teaching us how to apply a variety of CV algorithms in Python, using libraries like NumPy and OpenCV. Here is a quick overview of what each of these assignments related to:

Lab 1: Photometric Stereo & Colour. We had to implement the photometric stereo algorithm, calculate Albedo and Suface Normals and reconstruct images on a 3D surface. There were also tasks related to colour spaces, intrinsic image decomposition and colour constancy.

Lab 2: Neighbor Processing & Filters. We implemented low-level filters like Gaussian and Gabor filters. Applied denoising algorithms, for instance using neighborhood processing. Also implemented first- and second-order filters for edge detection and foreground-background seperation.

Lab 3: Harris Corner Detector & Optical Flow. We implemented the Harris Corner detection algorithm, which detects points of interest in an image. Then, we implemented the Lucas-Kanade optical flow algorithm, allowing us to track how the HCs change over sequences of images. Finally, we implemented the feature tracking algorithm.

Lab 4: Image Alignment and Stitching. We combined the insights of previous assignments and used RANSAC to estimate the best transformation between images. We then stitched two images together, using the aforementioned algorithms.

Lab 5: Part 1: Image Classification using Bag-of-words. We used SIFT descriptors to build a visual vocabulary, which we then utilized in an image classification schema on CIFAR-10. Part 2: CNNs for Image Classification. Similar task to Part 1, but now we trained a Convolutional Neural Network to classify CIFAR-10 images.
